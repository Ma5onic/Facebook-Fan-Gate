h1. Facebook Fan Gate

A really simple and quick to implement Facebook 'Fan/Like Gate' built using just a few lines of PHP.

h2. Installation

* Using the Developer App on Facebook, setup your new application and configure it's URLs to point at your host.
* Rename environment.php.default to environment.php and set the *facebook_app_id* to your Facebook Application ID
* Point apache at the root folder and you're good to go.
